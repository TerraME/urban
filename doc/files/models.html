UrbanSprawl



A Model that describes the corelation between urban sprawl and income segregation.

Arguments

  • alpha: The preference parameter for disposable income in Cobb-Douglas production function. The preference parameter for amount of living area is 1-Î±. It must be a value between 0 and 1, with default 0.5.
  • cityDimension: The city dimension. The default value is 25.
  • finalTime: The final time of the simulation. The default value is 50.
  • inequalityLevel: The predefined income distributions according to the inequality level. It must be a value between "low", "medium" or "high", with default "high".
  • initialArea: The initial/maximum available living area per cell. The default value is 100.
  • initialAvgGood: The average composite goods in the model. The default value is 400.
  • initialRent: The initial/minimum rent price per living unit. The default value is 10.
  • movingDiscount: The discount-rate to move to a new place. It must be a value between 0.1 and 1, with default 0.9.
  • pop: The initial population. The default value is 1000.
  • senseRadius: If this parameter is true, the sensing radius of households will be set proportionally to their income. The minimum radius will be 1 distant units. The default value is true.
  • stress: If this parameter is true, only households feeling stress will start to search for new places. Otherwise, all households will search for new places regardless their current state. The default value is true.
  • transportCost: The transport cost per unit. The range of value is from 1 to 50. The default value is 10.
